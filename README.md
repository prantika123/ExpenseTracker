This is simple Expense Tracker project developed using Python.
It allows users to record daily expenses, view all records, and calculate total spending.
Features-
1.Add new Expense
2.View all expenses
3.Calculate total expenses
4.Data stored in Database.
Technologies Used-
1.Python
2.SQL
How to Run:
1. Open the file in IDLE or any Python editor
2. Run the program: expense_tracker.py
4. Follow the menu instructions.
